# typescript-blue 💙

TypeScript's blue color package (with some logos 😎)

## Usage 🚀

```js
const blue = require('typescript-blue')
const logo = require('typescript-blue/logo')
const logoText = require('typescript-blue/logo-text')

// blue = '#007ACC'
// logo = '<svg>...</svg>'
```
